Block Elements

The elements which occupy the complete horizontal space and does not let any other element occupy their horizontal space. We can specify height and width of these elements. Ex - div, p

Inline Elements

These elements let other elements occupy their horizontal space amd does not let us to specify the height and width

Inline Block Elements

These elements have mixed properties of inline and block elements. They allow to use their horizontal space and also we can specify height and width
